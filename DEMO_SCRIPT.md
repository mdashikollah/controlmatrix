# 2:50 Demo Script

## 0:00–0:18 — Problem

“Operational work is often scattered across email, spreadsheets and individual memory. When a required document is deferred or an employee changes role, accountability and evidence can disappear.”

## 0:18–0:32 — Solution

“ControlMatrix is an offline-first task, project and deferral governance system with backup ownership, human-reviewed GPT-5.6 operational briefs and tamper-evident audit evidence.”

## 0:32–1:30 — Live workflow

- Press **Start 90-second judge demo**.
- Show the synthetic high-risk deferral.
- Press **GPT-5.6 Brief**.
- Confirm synthetic/non-confidential data.
- Generate the live brief.
- Point to recommended actions, missing information and risks.
- Edit one sentence.
- Choose **Edited before use**, enter a note and save.

Narration: “The model recommends; it cannot close the record, change risk or send email. A human must decide.”

## 1:30–1:55 — Evidence

- Open GPT-5.6 Operational Briefs.
- Show model, prompt version and reviewer decision.
- Open Audit Trail.

Narration: “Generation metadata and the reviewer outcome become part of the evidence chain.”

## 1:55–2:14 — Offline continuity

- Switch the browser offline.
- Open tasks and the deferral register.

Narration: “The operational workflow continues offline. Live GPT-5.6 generation waits until connectivity returns, so AI is not a single point of failure.”

## 2:14–2:38 — Codex

- Show the primary Codex session and a concise commit/change view.

Narration: “I used Codex to analyse the pre-existing codebase, implement the structured GPT-5.6 workflow and Cloudflare endpoint, improve the judge path, test failure states and document exactly what was added during Build Week.”

## 2:38–2:50 — Close

“Generic task managers track work. ControlMatrix turns deferred work into accountable action while preserving human authority.”
